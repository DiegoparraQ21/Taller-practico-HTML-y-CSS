Laboratorio del modelo de cajas en CSS

Contenido

Esta caja muestra unicamente el contenido con un tamaño fijo. El texto ocupa el espacio interno del elemento.

Padding

Esta caja tiene un relleno grande para evidenciar como el contenido se separa de los bordes.

Border

Esta caja posee un borde grueso y de color diferente para mostrar los limites del elemento.

Margin

Esta caja tiene un margen externo mayor, generando mas separacion respecto a los demas elementos.

Border Radius

Esta caja tiene esquinas redondeadas gracias a la propiedad border-radius.
